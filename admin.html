🔔 Supabase Auto-Ping

To prevent Supabase from pausing due to inactivity, set up a free auto-ping at cron-job.org:

1. Go to cron-job.org → Create free account
2. Click Create Cronjob
3. URL: YOUR_SUPABASE_URL/rest/v1/enquiries?select=id&limit=1
4. Schedule: Every 3 days
5. Save → Done! Supabase will never pause.
